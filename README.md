## 🔥 Featured Projects

### 🛡️ FinShield - AI Powered UPI Fraud Detection System

FinShield is a full-stack AI-powered platform that helps users detect suspicious UPI messages, scam links, UPI IDs, and payment screenshots before trusting them.

**Tech Stack:** React, Tailwind CSS, FastAPI, MongoDB, ML, OCR, JWT Auth

**Main Features:**
- UPI scam message detection  
- Trust Score and Risk Score  
- ML-based scam category prediction  
- URL and UPI ID checker  
- Screenshot OCR analysis  
- PDF scam report export  
- Emergency fraud guide  
- FinShield Assistant chatbot  

🔗 Repository: [FinShield](https://github.com/premarihar77/finshield)
